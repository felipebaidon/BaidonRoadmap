This assignment consist of creating a simple thread program which will output 
to Linux system log, the statements include debug messages from the program, timestamp and computer's information.

To create binary execute: "make pthread"
To clean previous binary execute: "make clean"

This program should be run on a Linux system, the output of the program should be located in /var/log/syslog, run "tail -f /var/log/syslog to see its output
